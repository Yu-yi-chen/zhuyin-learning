# Listen Quiz — Validation

## 驗收清單

### 題目與播放

- [ ] 進入 `/listen` 後約 700ms 自動播放一個符號發音
- [ ] 顯示 4 個選項按鈕
- [ ] 點擊播放按鈕可重播發音
- [ ] 每次重播同一題的發音不變

### Easy 模式答題

- [ ] 點擊正確選項 → 進入答對狀態，顯示 confetti + 蘋果 +10
- [ ] 點擊錯誤選項 → 該選項變紅且 disabled，其他選項仍可點擊
- [ ] 點錯後再點正確 → 進入答對狀態，**無** confetti（因為有過錯誤）
- [ ] 點擊「下一題 →」→ 新題生成，自動播放新題發音（約 300ms 後）
- [ ] 4 個選項都包含羅馬拼音

### Challenge 模式

- [ ] 切換到 Challenge → 當前題重置
- [ ] 未答對且無錯誤情況下，4 個選項**不顯示**羅馬拼音
- [ ] 答錯後，羅馬拼音**顯示**（`hadWrong = true`）
- [ ] 干擾選項優先來自相似組（如正確答案是ㄅ，其他選項可能是ㄆ ㄇ ㄈ）

### 積分

- [ ] 首次答對 → 積分 +10（Sidebar 顯示積分增加）
- [ ] 有錯再答對 → 積分不變

### 答對覆蓋層

- [ ] 覆蓋層出現後顯示正確符號的筆順動畫
- [ ] 首次答對有 confetti Lottie 動畫
- [ ] 筆順預覽 toggle 關閉時，覆蓋層內不顯示筆順動畫

### BGM

- [ ] 進入聽力頁後 BGM 靜音/停止
- [ ] 返回首頁後 BGM 重新播放（若原本未靜音）
- [ ] 全域靜音情況下進出，BGM 狀態不改變

### 難度持久化

- [ ] 選擇 Challenge 後重新整理，進入時仍是 Challenge 模式
- [ ] 筆順預覽 toggle 狀態重整後持久

## 已知限制

- 積分（`score`）目前有統計但無 UI 顯示（`score.correct / score.total`）
- Challenge 模式若相似組符號不足（如 ㄩ 所在組只有 3 個），剩餘選項隨機補充
