# Symbol Map — Validation

## 驗收清單

### 顯示

- [ ] 進入 `/map` 後顯示 37 個符號圓圈
- [ ] 每個圓圈同時可見注音字元和羅馬拼音
- [ ] Grid 進場時有 stagger 動畫效果
- [ ] Tab 列顯示四個選項，各有正確的數量標示（37 / 21 / 3 / 13）

### 過濾

- [ ] 點「聲母(21)」後只顯示 21 個聲母符號
- [ ] 點「介音(3)」後只顯示 ㄧ ㄨ ㄩ
- [ ] 點「韻母(13)」後只顯示 13 個韻母符號
- [ ] 切換 Tab 後 Grid 重新播放 stagger 動畫

### 完成狀態

- [ ] 完成一個練習後返回 `/map`，該符號顯示完成樣式
- [ ] 重新整理頁面後，已完成的符號仍顯示完成樣式
- [ ] 完成狀態在切換分類 Tab 後持續顯示

### 導航

- [ ] 點擊ㄅ → 跳轉到 `/practice/ㄅ`（或其 URL encode 形式）
- [ ] 頂部 Navbar 的返回按鈕 → 返回 `/`

### 音效

- [ ] 進入頁面時 BGM 開始播放（若未靜音）
- [ ] 全域靜音狀態下進入，BGM 保持靜音

## 已知限制

- 積分/完成狀態重整後仍在（localStorage），但積分進度（points/level）重整後歸零
- 符號 `dataIssue` 旗標（若設定）會在圓圈上顯示警示點，目前資料中無此情況
