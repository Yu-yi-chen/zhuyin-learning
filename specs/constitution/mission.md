# Mission

## 產品目標

BopoBear 是一個讓非母語學習者能夠快樂學習台灣注音符號（ㄅㄆㄇㄈ）的互動式網頁應用。

## 目標用戶

- 英語或日語背景的第二語言學習者
- 有基礎漢語知識，剛開始接觸注音拼音系統
- 主要在桌面瀏覽器使用

## 核心價值主張

| 痛點 | BopoBear 的解法 |
|------|----------------|
| 注音筆順難記 | 互動式筆順動畫示範 + 練習 |
| 發音難區分 | 聽音辨字測驗，即時音檔播放 |
| 學習無動力 | 積分、升等、完成徽章、confetti 動畫 |
| 記不住哪個學過 | 完成標記持久化到 localStorage |

## 設計原則

1. **樂趣優先**：每個正確回應都有動畫或音效回饋，學習要像遊戲
2. **即時回饋**：每個動作（答題、練習完成）立刻有視覺 + 聽覺回應
3. **零門檻入門**：不需登入即可使用所有學習功能
4. **單一真相**：所有符號資料集中在 `src/data/zhuyin.js`，改一處全站更新
5. **漸進挑戰**：Easy → Challenge 難度，讓用戶自然進步

## 不做什麼（Scope Boundaries）

- 不做聲調練習（注音本身不標調）
- 不做漢字聯繫（專注在注音符號本身）
- 不做行動裝置 native app（Web 優先）
- 不做用戶資料後端（積分只存 session，完成記錄存 localStorage）
